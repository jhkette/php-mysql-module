Description

This zip file contains the files necessary to complete the Manipulating Graphics Hands on exercise

------------
The permissions on the "thumbs" directory should be set to 777.
You can do this via FTP (or the command line if you prefer).

-------------
All configuration settings for this application can be found in:
includes/config.inc.php
